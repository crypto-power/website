<h1>Cryptopower Website</h1>

## Installation:
  - This application requires installation of npm as guided [here](https://docs.npmjs.com/cli/v9/commands/npm-install)
  - Install gulp cli globally using the following command:
  ```bash
  npm install gulp-cli -g
  ```
  - Run `npm install` to download the required node modules dependencies.
  - Run the website as:

